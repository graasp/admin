We're wrapping up the year with some small but useful updates to make your experience smoother. Also, please note the following notice:

> #### Deleting old items in your Trash {: .warning}
>
> Starting mid-January, items in the Trash that are older than 3 months will be permanently deleted. Please review or restore any important items before then.

Read on for the full list of changes.

<!-- Everything below this will not be shown in the post overview -->
<!-- truncate -->

## Changes

- Improved mobile experience by showing authentication buttons in the landing page menu
- Added notices for previously deleted items on home and trash pages
- Optimized item deletion performance with new database indexes

## Library

- Fixed OER reference links on the OER page
- Resolved a rare glitch when changing languages that could leave an incorrect language cookie
